# Flutter Packages Used

The following open-source packages are used in this project.

## just_audio

A powerful audio playback library for Flutter applications.

Used for:
- Playing local audio files
- Streaming audio
- Managing playback states

License: Apache 2.0

---

## on_audio_query

Provides access to audio files stored on the device.

Used for:
- Querying local songs
- Reading metadata such as title and artist
- Accessing device music libraries

---

## permission_handler

Handles runtime permissions across Android and other platforms.

Used for:
- Requesting audio/media access
- Managing permission states

---

## flutter_markdown

Renders Markdown files inside the Flutter application.

Used for:
- Displaying embedded documentation
- Rendering project policy pages

---

## marquee

Provides scrolling text widgets.

Used for:
- Displaying long song titles
- Creating animated text effects
